# Описание кнопок "Открыть модель" и "Полная модель" в карточке объекта

## 📍 Где находятся кнопки

Кнопки отображаются в карточке инвестиционной идеи (`TradeIdeaCard`) для **FREE пользователей** (не PRO).

---

## 🎯 Назначение кнопок

### Концепция монетизации:
- **FREE пользователи** видят только базовую информацию и уровень интереса (звезды 1-5)
- Для получения **полной финансовой модели** объекта нужно либо:
  1. Оформить **PRO подписку** (полный доступ ко всем объектам)
  2. Купить **разовую модель** за 199₽ (доступ к одному объекту)

---

## 🔍 Что видит FREE пользователь в карточке:

### 1. Базовая информация (всегда видна):
- ✅ Адрес объекта
- ✅ Площадь (м²)
- ✅ Количество комнат
- ✅ Цена объекта
- ✅ Бейдж "Ниже рынка на X%" или "Выше рынка на X%"
- ✅ **Уровень интереса** (1-5 звезд) - общая оценка привлекательности

### 2. Секция "Потенциал" (ограниченная):
```
┌─────────────────────────────────────┐
│ Уровень интереса: 4/5               │
│ [Открыть модель →]                  │
└─────────────────────────────────────┘
```

**Кнопка "Открыть модель →"** (внутри секции потенциала)
- Расположение: внутри блока с уровнем интереса
- Цена: не указана (ведет к выбору: PRO или разовая покупка)
- Назначение: открыть модальное окно с выбором:
  - Оформить PRO подписку
  - Купить разовую модель за 199₽

### 3. Кнопка "Полная модель 199 ₽" (внизу карточки):
```
┌─────────────────────────────────────┐
│ [Сохранить] [Полная модель 199 ₽]   │
└─────────────────────────────────────┘
```

**Кнопка "Полная модель 199 ₽"**
- Расположение: внизу карточки, рядом с кнопкой "Сохранить"
- Цена: 199₽ (явно указана)
- Назначение: сразу купить разовую модель объекта

---

## 💎 Что видит PRO пользователь:

PRO пользователи видят **всю информацию без ограничений**:

### 1. Полная информация о потенциале:
- **Для стратегии "Флип":**
  - Потенциал флипа: 500 000 ₽ (14.3%)
  - Оценка маржи после ремонта
  - Прибыльность сделки

- **Для стратегии "Аренда":**
  - Доходность: 11.2% годовых
  - Ожидаемая аренда: ~35 000 ₽/мес
  - Окупаемость

### 2. Риски:
- ⚠️ Объект дороже рынка на 12%
- ⚡ Объект на рынке более 90 дней
- 📉 Снижение цены за 3 месяца на 5%

### 3. Без кнопок "Открыть модель":
- PRO пользователи не видят эти кнопки
- Вся информация доступна сразу

## 📄 Что включает "Полная модель" (после оплаты):

После оплаты 199₽ пользователь получает доступ к:

### 1. Детальный расчет потенциала:

**Для стратегии "Флип":**
```
Потенциал флипа:
├─ Текущая цена: 3 500 000 ₽
├─ Рыночная цена: 4 100 000 ₽
├─ Скидка: 600 000 ₽ (14.6%)
├─ Стоимость ремонта: 525 000 ₽ (15%)
├─ Общие вложения: 4 025 000 ₽
├─ Оценочная цена после ремонта: 4 510 000 ₽
└─ Потенциальная маржа: 485 000 ₽ (12.0%)
```

**Для стратегии "Аренда":**
```
Доходность:
├─ Цена объекта: 3 500 000 ₽
├─ Ожидаемая аренда: 35 000 ₽/мес
├─ Годовая аренда: 420 000 ₽
├─ Доходность: 12.0% годовых
├─ Окупаемость: 8.3 года
└─ Цена за м²: 76 923 ₽
```

### 2. Анализ рисков:
```
Риски:
├─ ⚠️ ВЫСОКИЙ: Объект дороже рынка на 12%
├─ ⚡ СРЕДНИЙ: Объект на рынке более 90 дней
└─ ℹ️ НИЗКИЙ: Возможен старый дом
```

### 3. Финансовая модель:
- Таблица cash-flow (если стратегия аренда)
- График окупаемости
- Сценарии (оптимистичный, реалистичный, пессимистичный)

### 4. Рекомендации:
- Подходит ли объект для выбранной стратегии
- Что нужно проверить перед покупкой
- Оценка ликвидности

### 5. Основание расчетов:
- По каким аналогам считали
- Источники данных
- Период анализа

## 🔧 Техническая реализация:

### Backend должен предоставить:

1. **Endpoint для разовой покупки:**
   ```
   POST /api/invest/trade-ideas/{dealId}/unlock
   ```
   - Создает платеж на 199₽
   - Возвращает payment URL
   - После оплаты открывает доступ к полной модели

2. **Endpoint для получения полной модели:**
   ```
   GET /api/invest/trade-ideas/{dealId}/full-model
   ```
   - Проверяет права доступа (PRO или оплаченная модель)
   - Возвращает полные данные:
     - Детальный потенциал
     - Риски
     - Финансовая модель
     - Рекомендации

3. **Проверка доступа:**
   - PRO пользователи: всегда доступ
   - FREE пользователи: проверка оплаты конкретного объекта

---

## 📝 Итог:

**"Открыть модель →"** - кнопка для выбора способа получения доступа (PRO или разовая покупка)

**"Полная модель 199 ₽"** - кнопка для быстрой разовой покупки модели конкретного объекта

**Полная модель** включает:
- Детальные расчеты потенциала
- Анализ рисков
- Финансовую модель
- Рекомендации
- Основание расчетов
