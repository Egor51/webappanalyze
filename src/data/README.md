# Тестовые данные для сервиса инвестиций

## Описание

Файл `investmentTestData.js` содержит тестовые данные для разработки и тестирования сервиса инвестиций без необходимости подключения к реальному API.

## Структура данных

### Варианты инвестиций (mockInvestmentOptions)

Массив объектов с информацией об инвестиционных вариантах:

```javascript
{
  id: number,                    // Уникальный идентификатор
  address: string,               // Полный адрес объекта
  location: string,              // Альтернативное поле для адреса
  type: string,                  // Тип объекта (Квартира, Дом, Район)
  price: number,                 // Цена объекта в рублях
  expectedReturn: number,        // Ожидаемая доходность в процентах
  growth: number,                // Рост цены за год в процентах
  area: number,                  // Площадь (для квартир/домов)
  floor: number,                 // Этаж (для квартир)
  totalFloors: number,           // Всего этажей в доме
  year: number,                  // Год постройки
  description: string,           // Описание объекта
  // Дополнительные поля для районов:
  averagePrice: number,          // Средняя цена в районе
  propertiesCount: number,       // Количество объектов в районе
  // Дополнительные поля для домов:
  landArea: number              // Площадь участка
}
```

### Лучшие варианты (mockBestOptions)

Массив объектов с лучшими инвестиционными вариантами. Структура аналогична `mockInvestmentOptions`, но дополнительно включает:

```javascript
{
  rank: number,                  // Рейтинг (1-10)
  score: number                  // Оценочный балл (0-100)
}
```

## Функции

### `getMockInvestmentOptions(amount)`

Возвращает варианты инвестиций, подходящие под указанную сумму.

**Параметры:**
- `amount` (number) - Сумма инвестиций в рублях

**Возвращает:**
- Массив объектов, отсортированных по доходности (по убыванию)

**Логика фильтрации:**
- Выбираются варианты в диапазоне ±20% от указанной суммы
- Сортировка по `expectedReturn` (по убыванию)

### `getMockBestOptions()`

Возвращает топ-10 лучших инвестиционных вариантов.

**Возвращает:**
- Массив из 10 объектов, отсортированных по рейтингу

### `getMockOptionById(id)`

Возвращает конкретный вариант по его ID.

**Параметры:**
- `id` (number) - ID варианта

**Возвращает:**
- Объект варианта или `null`, если не найден

## Использование в компоненте Investing

Компонент `Investing.jsx` использует тестовые данные через флаг `USE_MOCK_DATA`:

```javascript
const USE_MOCK_DATA = true // true - использовать тестовые данные, false - реальный API
```

При `USE_MOCK_DATA = true`:
- Все запросы используют тестовые данные
- Добавлена имитация задержки сети (500-800ms)
- При ошибках API автоматически используется fallback на тестовые данные

## Примеры использования

### Получить варианты для суммы 5 000 000 рублей:

```javascript
import { getMockInvestmentOptions } from '../data/investmentTestData'

const options = getMockInvestmentOptions(5000000)
console.log(options) // Массив подходящих вариантов
```

### Получить лучшие варианты:

```javascript
import { getMockBestOptions } from '../data/investmentTestData'

const bestOptions = getMockBestOptions()
console.log(bestOptions) // Топ-10 вариантов
```

## Данные в файле

Файл содержит:
- **10 вариантов** для поиска по сумме
- **10 лучших вариантов** (топ-10)
- Разнообразные типы объектов: квартиры (1-3 комнаты), дома, районы
- Реалистичные цены от 2.8 до 8.5 млн рублей
- Доходность от 10.5% до 16.5%
- Рост цен от 6.8% до 11.2%

## Переход на реальный API

Когда API будет готов, установите:

```javascript
const USE_MOCK_DATA = false
```

Компонент автоматически начнет использовать реальные endpoints:
- `GET /ads/investment/options?amount={amount}`
- `GET /ads/investment/best`
